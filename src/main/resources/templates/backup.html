💾 Архив истории отчётов

Компактный снимок прогонов и сообщений · не полная копия диска H2

msg

Зачем это нужно

  • Сохранить историю автотестов в лёгком формате history-*.json.gz (JSON + сжатие).
  • При сбое или миграции — восстановить данные из архива без перезапуска сервера.
  • Плановый архив раз в неделю и только если данные изменились — без лишней нагрузки.

Настройки

  • Статус: вкл
  • Каталог: ./backups
  • Расписание: cron
  • Хранить архивов: 8
  • Пропуск без изменений: да

Перед восстановлением из списка создаётся страховочный архив текущих данных (если они есть).

Загрузить архив с компьютера

Форматы: history-*.json.gz (как при скачивании), распакованный .json или .zip. Не переименовывайте и не сохраняйте пустой файл — иначе восстановление удалит все отчёты. Без галочки файл только появится в списке архивов на стенде — восстановить можно позже кнопкой «Восстановить».
